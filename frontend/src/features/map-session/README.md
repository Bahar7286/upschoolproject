# map-session

Harita sayfası oturum state:

- `use-map-filters` — kategori, yarıçap, POI görünürlüğü
- `use-map-session` — şehir/ilçe/geo merkez çözümlemesi
- `use-active-route-map` — geofence, sesli anlatım, konum izleme, rota tamamlama
- `MapControlsPanel` — kategori filtreleri
- `MapActiveRouteSection` — aktif rota kontrolleri ve navigasyon paneli
